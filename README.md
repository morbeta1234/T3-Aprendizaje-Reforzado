# t3-reforzado

## Setup
Antes que nada, se recomienda setear un ambiente virtual. Por ejemplo,
```bash
python -m venv .env
source .env/bin/activate
```
Después, instalar las dependencias:
```bash
pip install -r requirements.txt
```
